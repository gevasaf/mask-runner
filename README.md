# Mask Runner

A mini endless-runner game made during a **48-hour game jam**. **No commercial intentions** — just for fun and learning.

## What it is

- **Genre:** Endless runner
- **Platform:** Unity
- **Controls:** Swipe (or keys) to move between lanes; avoid enemies, collect coins and power-ups.
- **Features:** Multiple worlds (desert, lava, space, underwater) that change as you run, power-ups (speed boost, invincibility), and simple story beats.

## Running the game

1. Open the project in **Unity** (folder: `My project`).
2. Open the main scene and press Play.

## Project layout

- **My project/** — Unity project (scenes, assets, scripts).
- **concept/** — Concept art and world ideas.
- **runway_segments/** — Video segments used in the game.
- **scripts/** — Python tools for generating/processing Runway ML video content (optional, not required to play).
